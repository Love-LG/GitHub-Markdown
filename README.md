# GitHub-Markdown
GitHub常用的markdown语法

### markdown常用列表
无序列表使用`*`语法如下：
```bash
* item1
* item2
* item3
  * item3_1
  * item3_2
* item4
```

  有序列表的使用语法如下：
  ```bash
  1. item1
  2. item2
  3. item3
     1. item3_1
     2. item3_2
  4. item4
  ```

### markdown标题
markdown常用标题使用`#`语法如下：
```bash
# this is <h1> tag
## this is <h2> tag
### this is <h3> tag
###### this is <h6> tag
```

 <summary>Example</summary>

## 这是二级标题
### 这是三级标题
 
 </details>
